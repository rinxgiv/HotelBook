# Zadani - HotelBook

HotelBook je frontend aplikace pro vyhledavani hotelu a rezervaci pokoju.
Bezi cele v prohlizeci jako PWA, bez vlastniho backendu.

## Funkce

- hledani hotelu podle mesta
- prime REST API volani Google Places API
- detail hotelu
- rezervace pokoju
- prehled a ruseni rezervaci
- `localStorage`
- PWA

## Technologie

- HTML
- CSS
- JavaScript
- Google Places API
- localStorage
- Service Worker
